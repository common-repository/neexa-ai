== Description ==

This plugin seamlessly integrates Neexa.AI's 24/7 AI Powered Sales Agent/Assistant onto any WordPress site. Please note that the functionality of this plugin relies on Neexa.AI, a third-party service.

**Disclosure:**
By using this plugin, you acknowledge and agree that certain data may be transmitted to Neexa.AI for the proper functioning of the AI Sales Agent/Assistant. This includes but is not limited to user interactions, conversations, and related data.
